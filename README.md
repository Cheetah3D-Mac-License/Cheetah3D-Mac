## How to Install

1. Download the latest version of Cheetah3D for macOS.  
2. Open the DMG installer and move Cheetah3D.app to the Applications folder.  
3. Launch the app and allow it through macOS Gatekeeper if required.

## Overview

<a href="#">Cheetah3D for macOS</a> is a compact 3D creation suite that emphasises approachability without discarding power. It offers modelling, rigging, animation and rendering tools inside a single, coherent interface tailored to the Mac. Artists can shape meshes, apply materials and light scenes with a focus on visual feedback rather than endless panels. For individuals and small teams, this balance makes it possible to complete meaningful 3D work without assembling a large pipeline.

The toolset includes polygon modelling, subdivision surfaces and <a href="#">procedural modifiers</a> that help build clean, flexible geometry. Bones and IK systems support character work, while timeline and curve editors bring animation under precise control. Built-in rendering options allow scenes to be output with lighting and shading that match the creative intent, and export paths support moving assets into game engines or compositing tools when needed. Scriptability and extensibility give more technical users room to customise behaviours.

On macOS, Cheetah3D benefits from graphics capabilities, high-resolution displays and input devices that feel natural during modelling. It can run comfortably alongside texture tools, reference boards and version control systems. Over long projects, it becomes a reliable workhorse for the 3D side of a creative practice, handling everything from quick concept sculpts to finished shots.

## Major Features
- Polygon and subdivision surface modelling tools  
- Rigging and IK systems for character animation  
- Keyframe and curve-based animation editors  
- Integrated rendering engine for stills and sequences  
- Material and lighting controls for look development  
- Support for importing and exporting common 3D formats  
- Scriptability for automating repetitive tasks  
- macOS-native interface optimised for 3D workflows  
- Suitable for illustration, motion graphics and games  
- Lightweight enough for indie and solo creators  
- Helps artists move from concept to finished renders  
- Built as an all-in-one 3D environment for the Mac
